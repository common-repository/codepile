With an assortment of simple and useful features that can be enabled as you like.

Configure your site how you want it without having to install or code plugins for every little thing.

== Installation ==
= From your WP admin dashboard =

1. Go to Plugins > Add New
2. Search for CodePile
3. Click Install and then click Activate

= Download from wordpress.org =

1. Download the zip file from [wordpress.org](https://wordpress.org/plugins/codepile "wordpress.org/plugins/codepile")
2. Go to your WP admin and go to Plugins > Add New > Upload Plugin
3. Select the file you downloaded and click Install Now
4. Activate CodePile

= After activation =

Go to your WP admin to CodePile > Features and enable any features you want. Each feature you enable will add a new menu item under the CodePile menu with more options.
